# ☁️ What is AWS?

Amazon Web Services (AWS) is a comprehensive cloud computing platform provided by Amazon that offers scalable, reliable, and cost-effective IT infrastructure over the internet.


---

## 📌 Explanation

Instead of purchasing physical servers, organizations can use AWS services such as:

- Computing power  
- Storage  
- Databases  
- Networking  
- Analytics  
- Security  
- Machine Learning  
- Application deployment  

All services are available **on demand**, meaning they can be accessed anytime over the internet.


---

## 💰 Pricing Model

AWS follows a **Pay-As-You-Go** pricing model.

This means:

- Users pay only for the resources they use  
- No upfront investment is required  
- It is cost-efficient for startups and enterprises  


---

## 🎯 Key Benefits

- Scalability → Increase or decrease resources anytime  
- High reliability → Strong global infrastructure  
- Cost-effective → Pay only for usage  
- Flexibility → Wide range of services  
- No physical hardware required  


---

## 📌 One-Line Definition (For Exams/Interviews)

AWS is a cloud computing platform that provides on-demand IT resources over the internet with a pay-as-you-go pricing model.

# ☁️ AWS Cloud Overview

AWS operates one of the largest cloud infrastructures in the world.


---

## 🌍 Key Components

### 1️⃣ Regions

Geographic locations worldwide where AWS data centers exist.  

Examples: Mumbai, Tokyo, London, N. Virginia  

Each region operates independently for better reliability and compliance.

**Benefits:**

- Low latency  
- Data residency control  
- High availability  


---

### 2️⃣ Availability Zones (AZs)

Multiple isolated data centers inside a region.

Each AZ has:

- Independent power  
- Cooling systems  
- Networking infrastructure  

**Purpose:** Ensures fault tolerance and disaster recovery.


---

### 3️⃣ Edge Locations

Used for content delivery via CDN (Amazon CloudFront).  

Stores cached data closer to users.  

**Benefit:** Improves website and application speed globally.


---

### 4️⃣ Local Zones & Outposts

Bring AWS services closer to end users.

Designed for:

- Gaming  
- Streaming  
- Real-time analytics  


---

# ☁️ AWS Cloud Service Categories


---

## 1️⃣ Compute Services

Provide processing power for running applications.

- **Amazon EC2**  
  Virtual servers with full OS control  

- **AWS Lambda**  
  Serverless computing (no server management)  

- **Elastic Beanstalk**  
  Easy application deployment  

- **Amazon ECS / EKS**  
  Container management (Docker & Kubernetes)  


---

## 2️⃣ Storage Services

Store files, backups, and application data.

- **Amazon S3**  
  Scalable object storage  

- **Amazon EBS**  
  Block storage for EC2  

- **Amazon Glacier**  
  Low-cost archival storage  

- **AWS Storage Gateway**  
  Hybrid storage solution  


---

## 3️⃣ Database Services

Managed databases without manual work.

- **Amazon RDS**  
  Relational databases  

- **Amazon DynamoDB**  
  NoSQL database  

- **Amazon Redshift**  
  Data warehouse  

- **Amazon Aurora**  
  High-performance database  


---

## 4️⃣ Networking & Content Delivery

Handles connectivity and traffic routing.

- **Amazon VPC**  
  Private cloud network  

- **Elastic Load Balancer (ELB)**  
  Traffic distribution  

- **Amazon Route 53**  
  DNS management  

- **AWS CloudFront**  
  Global CDN  


---

## 5️⃣ Security & Identity Services

Protect resources and manage access.

- **AWS IAM**  
  User roles and permissions  

- **AWS Shield**  
  DDoS protection  

- **AWS WAF**  
  Web firewall  

- **AWS KMS**  
  Encryption management  


---

## 6️⃣ Monitoring & Management Services

Track and manage infrastructure.

- **Amazon CloudWatch**  
  Logs and monitoring  

- **AWS CloudTrail**  
  Activity tracking  

- **AWS Config**  
  Resource monitoring  

- **AWS Systems Manager**  
  Automation  


---

## 7️⃣ Analytics & Machine Learning

Process and analyze data.

- **Amazon Athena**  
  Query data from S3  

- **AWS Glue**  
  Data integration (ETL)  

- **Amazon SageMaker**  
  Machine learning  

- **Amazon QuickSight**  
  Data visualization  


---

## 8️⃣ Developer & DevOps Tools

CI/CD and automation tools.

- **AWS CodeCommit**  
  Source control  

- **AWS CodeBuild**  
  Build automation  

- **AWS CodeDeploy**  
  Deployment  

- **AWS CodePipeline**  
  CI/CD pipeline  


---

## 🔐 AWS Shared Responsibility Model

| AWS Responsibility        | Customer Responsibility      |
|--------------------------|-----------------------------|
| Physical data centers    | Application security        |
| Hardware & networking    | Data protection             |
| Global infrastructure    | Access management           |
| Cloud platform security  | OS & configurations         |
